## 💡 Tips & Tricks

### Development
- Use **Live Server** VS Code extension for instant preview
- Open **DevTools** (F12) to debug
- Use **mobile view** (F12 → phone icon) to test responsive
- Clear cache with **Ctrl+Shift+Del**

### Customization
- Search for text to find all instances
- Use Find & Replace (Ctrl+H) for bulk changes
- Keep backups before major edits
- Test changes immediately

### Deployment
