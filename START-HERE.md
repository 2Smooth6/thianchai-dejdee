## 🎓 Customization Examples

### Change Your Name
```html
<!-- Find this (around line 32) -->
<h1 class="hero-title">THIANCHAI DEJDEE</h1>

<!-- Change to -->
<h1 class="hero-title">YOUR NAME</h1>
```

### Update Your Email
```
Find: contact@thianchai.com
Replace with: your.email@company.com
```

### Change Primary Color
```css
/* In styles.css - Line 5 */
--color-accent: #00D4FF;  /* Change this to your color */
```

### Update Skill Proficiency
```html
<!-- Find skill progress bar -->
<div class="skill-progress" style="width: 85%"></div>

<!-- Change 85% to your percentage -->
<div class="skill-progress" style="width: 90%"></div>
```
