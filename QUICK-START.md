# Quick Start Guide

Welcome to your premium portfolio website! This guide will get you started in 5 minutes.

---

## Step 1: Preview Your Portfolio (2 minutes)

### Option A: Chrome/Edge/Firefox
1. **Open** the `portfolio` folder
2. **Right-click** `index.html`
3. **Select** "Open with" → Your browser
4. **Enjoy!** 🎉

### Option B: Visual Studio Code
1. **Open** the portfolio folder in VS Code
2. **Install** "Live Server" extension
3. **Right-click** `index.html`
4. **Select** "Open with Live Server"
5. **Automatic refresh** as you edit!

### Option C: Python (Any OS)
```bash
cd portfolio
python -m http.server 8000
# Visit: http://localhost:8000
```

## Quick Edits

### Change Colors
Open `styles.css` and update `:root` variables (lines 1-20):

```css
:root {
    --color-accent: #00D4FF;        /* Change this for primary color */
    --color-accent-secondary: #8B5CF6; /* Change this for secondary color */
}
```

[Color Picker Tool](https://htmlcolorcodes.com/)

### Add/Remove Projects
Find "PROJECTS PAGE" section in `index.html` (~line 480)
Copy/paste/edit project cards as needed.

### Update Skills
Find "SKILLS & TOOLS" section in `index.html` (~line 330)

## Troubleshooting

### Styles not showing?
1. Hard refresh: `Ctrl+Shift+R` (Windows) or `Cmd+Shift+R` (Mac)
2. Clear browser cache
3. Check file paths in HTML

### Animations not working?
1. Check if JavaScript is enabled
2. Try a different browser
3. Check browser console for errors (F12)

### Mobile looks broken?
1. Try a different phone/tablet
2. Test in Chrome DevTools (F12 → mobile icon)
3. Check viewport meta tag in HTML
